# Tiny Room · 2024.10

**Tiny Room**은 현대 오토에버 모빌리티 SW 스쿨에서 진행한 블로그 프로젝트로, 유저가 자신의 방을 꾸밀 수 있는 기능을 가지고 있는 블로그 서비스입니다.

## 🛠 프로젝트 기술 스택

- **프론트엔드**: React.js, styled components, MUI, axios, dayjs, zustand
- **백엔드**: Spring Boot, Spring Data JPA, MySQL, QueryDSL
